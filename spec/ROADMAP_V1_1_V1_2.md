# FastScript v1.1 / v1.2 Roadmap

## v1.1 (Target: 2026-07-15)

- Parser + AST + strict/lenient compiler modes
Owner: Compiler Team
- Typecheck/format/lint command suite
Owner: DX Team
- Nested layouts, route groups, parallel routes, streaming SSR
Owner: Runtime Team
- Security hardening (session policy, distributed limits, quota/abuse)
Owner: Security Team
- VS Code LSP feature expansion + diagnostics
Owner: Tooling Team

## v1.2 (Target: 2026-10-30)

- SSG + ISR + cache tags + invalidation
Owner: Runtime Team
- Production adapters (Vercel/Cloudflare) + edge parity
Owner: Deploy Team
- Data plane parity (Postgres, rollback tooling, queue backends, retention)
Owner: Platform Team
- Docs/versioning/playground/API reference/migration packs
Owner: Docs Team
- Ecosystem loop (templates, marketplace flow, KPI dashboards)
Owner: PM + Community

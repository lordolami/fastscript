# Real-World Adoption

FastScript `3.0.1` is the first real-world adoption line.

Use `/docs/adoption` as the canonical path for:

- starting a new `.fs` app with `fastscript create`
- migrating existing JS/TS incrementally with `npm run migrate -- app --dry-run`
- checking `/docs/support` before promising a framework/runtime pattern
- reporting valid JS/TS failures in `.fs` as compatibility bugs

The governed support matrix is the source of truth for what is currently `proven`, `partial`, `planned`, or `blocked`.
